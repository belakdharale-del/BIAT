Sélection stratégique : LightGBM

Le modèle LightGBM a été retenu comme modèle champion pour la surveillance du risque BIAT. Son algorithme GOSS (Gradient-based One-Side Sampling) permet de gérer efficacement les données déséquilibrées typiques du risque bancaire.

Dans ce contexte, le Recall est la métrique prioritaire. Un faux négatif (client risqué non détecté) représente un coût institutionnel bien supérieur à un faux positif (client sain contacté par erreur). LightGBM atteint un Recall de 0,9372 tout en maintenant une Accuracy de 97,36%.
